Container 

Los contenedores son el elemento de diseño más básico en Bootstrap y son necesarios cuando se utiliza nuestro sistema de cuadrícula predeterminado . Los contenedores se usan para contener, rellenar y (a veces) centrar el contenido dentro de ellos.

Responsive Image 

Crea imágenes responsivas añadiendo una clase .img-responsive a la etiqueta <img>. La imagen se adaptará bien al elemento principal.

1- .btn 2- .btn-default 3- .btn-primary 4- .btn-success 5- .btn-info 6- .btn-warning 7- .btn-danger 8- .btn-link

Navbar 

Con Bootstrap, una barra de navegación puede extenderse o colapsarse, dependiendo del tamaño de la pantalla. Una barra de navegación estándar se crea con <nav class="navbar navbar-default">.

Button Group 

Bootstrap permite agrupar una serie de botones (en una sola línea) en un grupo de botones. Utiliza un elemento <div> con clase .btn-group para crear un grupo de botones

Dropdown 

Un menú desplegable es un menú conmutable que permite al usuario elegir un valor de una lista predefinida
